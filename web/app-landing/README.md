# SGQ App 介紹網站（app.sagp-qp.com）

靜態落地頁：App 功能介紹、Play Store / App Store 連結、教師匯出入口。

**部署教學**：[從頭部署-app.sagp-qp.com.md](./從頭部署-app.sagp-qp.com.md)

## 本機預覽

雙擊 `本機預覽.bat` → http://localhost:8081

## 設定商店連結

編輯 `config.js`（範例見 `config.example.js`），再執行 `run-app-domain.bat` 上傳。

## AWS 部署

雙擊 `run-app-domain.bat`
